【hexo】主题美化-butterfly

hexo 官网有很多主题，其中个人比较喜欢的两个主题是 next 和 butterfly，这篇博客介绍 butterfly 主题的基本配置和美化。

教程主要参考博客：hexo博客butterfly主题美化。
主题的 Github 链接： Butterfly。

安装主题

  1. 新建博客框架，参看【hexo】博客搭建。
  2. 进入博客目录，下载主题
    方法一，用 git 将主题下载到 themes 目录 方法二，用 npm 将主题下载到 node_modules 目录
  3. 修改 _config.yml ，将其中 theme 参数改为 butterfly
  4. 安装渲染工具
  5. 博客目录下，新建文件 _config.butterfly.yml 并将主题中的 _config.yml 内容复制过来。

此时，用 hexo s 启动，就进入了 butterfly 主题。

注1：两种方法，主题的配置文件放在不同目录上。
注2：命令 npm i 中，i 为 install 的缩写，但也有差异，参见npm i和npm install的区别。npm i 默认为局部安装，安装目录为 node_modules。
注3：生成网页时，会将博客目录的 _config.yml 文件与主题目录的 _config.yml 合并，当主目录中存在 _config.主题名.yml 文件时，系统会优先使用该文件。

更多功能

_config.butterfly.yml 中还有很多功能设置，例如访问验证 site_verification，数据分析 Analysis，广告 Advertisement；此外还有主题配色 theme_color，背景动态等等效果设置。

踩坑的地方

一些样式失效，例如代码的mac样式，以及复制按钮。可能因为 _config.butterfly.yml 中一些参数填写错误。例如下边 author 参数只接收 true 和 false ，但填写了人名就会导致错误。
