函数说明

示例用了函数 latex_table，输入数据为 Python 列表，输出 tex 格式的表格。
函数的编写基于 latex 表格语法：

  • \begin{table} 表格环境
  • \begin{tabular} 表格内容
  • \scalebox{<放缩比例>} 表格较大时，用于缩小
  • \hline 表格行线
  • \rowcolor{gray!50} 设置行颜色，!50 表示 50%
  • & 表格内容对齐符号
