【hexo】博客搭建

一直想搭个一个自己的博客平台，一边学习，一边输出，加深记忆也方便查阅。在B站搜索了许多教程，最后选择用 hexo 来搭建。

参考链接

羊哥的B站视频：手把手教你从0开始搭建自己的个人博客。

搭建准备：

  • Win/linux/mac 系统
  • github 账号

个人常用设备为 Win10 + Ubuntu 双系统，并在 Ubuntu 系统中搭建博客；另外宿舍电脑用的 Win10，也搭建了博客，同步管理。不同系统的搭建过程基本一致。

软件安装

  1. 在官网下载安装 Nodejs，里边包含了包管理器 npm
  2. 用 npm 安装 cnpm
  3. 再用 cnpm 全局安装 hexo-cli

github 准备

  1. 注册 Github 账号
  2. 新建公开仓库 username.github.io # 这里 username 取自己的Github用户名。
  3. 配置 SSH 密钥，具体参考另一篇博客Github多账户设置。

踩坑的地方

  1. 部署显示 permission denied!
    • 这可能是 hexo 链接 Github 时，验证 ssh 密钥用的是根目录，解决方法是将用户目录 ~/ 下的 .ssh 文件夹复制到根目录。
    • 参考教程：部署Hexo显示Permission Denied的解决方法。
  2. 写博客如果直接新建空白 markdown 文件，要注意文件开头格式，否则可能启动失败。

博客框架搭建后，要修改配置文件，把模板的信息改为个人信息，同时可以修改主题样式，具体参看博客 【hexo】主题美化-butterfly。

用 hexo 搭建博客的好处在，后续编写文章只需要新建和编辑 Markdown 文件，而 Markdown 的语法比 html 和 latex 要简单得多，几分钟就能上手。这样可以将精力集中在内容写作上，而不是各种样式排版，参看Markdown 语法。
