markdown 学习笔记

Markdown 是一种纯文本格式的标记语言，通过简单的标记语法，可以使普通文本内容具有一定的格式。优点主要有：纯文本编辑，操作简单，支持内嵌 html 等。

参考简书：Markdown基本语法。（其中介绍代码高亮的部分有误， ``` 不需要加括号。）
