使用技巧

屏幕截图

默认方式：键盘按键

  1. Ctrl + PrtSc 截全屏
  2. Ctrl + shift + PrtSc 选择截屏区域

说明：

  • 按键 PrtSc 一般在键盘右上角的位置；
  • 默认截屏方式，图片保存在 ~/picture 目录下；
  • 截屏后，文件不会保存到剪贴板

高级操作：脚本+快捷键
