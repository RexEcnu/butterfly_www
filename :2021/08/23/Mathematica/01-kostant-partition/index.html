简介

本篇博客整理关于李代数 Kostant partition 函数的几种算法，以 C 型根系的计算为例，其他几个类型后续补充。

由于 Markdown 的代码块样式不支持 Mathematica，所以相关代码用截图的方式。

另外，Dynkin 图相关的 latex 还为编写，所以暂时留空。

公式及代码

连续根 $C_{l,r}$

设 C 型单李代数的 Dynkin 图为 $\alpha_0=>=\alpha_1==\cdots==\alpha_n$，其中 $\{\alpha_i\}_{i=0}^n$ 为相应的单根。
定义正根 $\beta_{l,r}:= \alpha_l+\alpha_{l-1}+\cdots+\alpha_1+\alpha_0+\alpha_1+\cdots+\alpha_r$
记 $C_{l,r}=\mathfrak{P}(\beta_{l,r}), l,r\geq 0$，其中 $\mathfrak{P}$ 为 Kostant partition 函数。
