【hexo】博客搭建

一直想搭个博客平台，一边学习，一边输出，加深记忆也方便查阅，在B站搜索了许多教程，最后决定用 hexo 来搭建。本篇介绍博客框架的搭建。

参考链接

羊哥的B站视频：手把手教你从0开始搭建自己的个人博客。

搭建准备：

  • Win/linux/mac 系统
  • github 账号

个人用的是 Win10 + Ubuntu 双系统，并在 Ubuntu 系统中搭建博客；另外宿舍电脑用的是 Win10，也搭建了博客，不同系统的搭建过程基本一致。博客可以在多个设备上同步管理，方法参考博客 CSDN:hexo多终端同步和管理。

软件安装

  1. 在官网下载安装 Nodejs，里边包含了包管理器 npm
  2. 用 npm 安装 cnpm
  3. 再用 cnpm 全局安装 hexo-cli

github 准备

  1. 注册 Github 账号
  2. 新建公开仓库 username.github.io # 这里 username 必须是自己的 Github 用户名。
  3. 由于本地 Git 仓库和 GitHub 仓库之间的传输是通过 SSH 加密的，所以需要一些设置，参考博客 Github 多账户设置及 git clone 加速。
