安装主题

  1. 新建博客目录，进入超级用户模式，输入代码初始化
  2. 进入博客目录，下载主题
    方法一，用 git 将主题下载到 themes 目录 方法二，用 npm 将主题下载到 node_modules 目录
  3. 修改 _config.yml ，将其中 theme 参数改为 butterfly
  4. 安装渲染工具

此时，用 hexo s 启动，就进入了 butterfly 主题。

注1：两种方法下载的主题，放在不同目录上，后续美化操作要进入相应目录。

注2：下载主题时，建议退出管理员权限，这样在编辑样式时，不用每次都进入超级管理员模式。

注3：命令 npm i 中，i 为 install 的缩写，但也有差异，参见npm i和npm install的区别。默认为局部安装，也即安装在 node_modules 目录。

找到 butterfly 主题文件夹，下边操作大多数是修改该文件夹下的 _config.yml。
